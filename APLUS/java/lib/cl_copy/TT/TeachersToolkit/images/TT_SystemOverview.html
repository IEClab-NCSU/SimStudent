Welcome to Teacher's Toolkit

Teacher's Toolkit provides an easy to use interface for performing administrative tasks associated with the Cognitive Tutor programs. Such tasks typically include:

  • Creating new class rosters at the beginning of a term to generate the necessary files for students to use the tutor.

  • Tracking student progress as they work through the tutor curriculum and printing class summary or individual progress reports.

  • Maintaining class rosters by changing student names, transferring students to different classes, or removing students.

  • Restarting a problem, skipping to the next problem, or changing the placement within the curriculum for a student.

  • Deleting class rosters once the term is over.

The Teacher's Toolkit screen interface is divided into the four areas illustrated above. These areas include:

Title Bar

This area displays the name of the screen that is currently displayed. For example, if you are creating a class roster, you would see "Teacher's Toolkit: Create Class" the title bar.

Task List Panel

This area on the left side of the screen displays all the tasks that you can perform using Teacher's Toolkit. There are four task categories:

  • Class Roster Maintenance
  • Curriculum-Related Tasks
  • Instructor Tasks
  • Reports

Each task category contains a list of associated tasks. You can expand (open) or contract (close) a category list by clicking on the arrow to the left of the category name, or on the name itself. If you move the mouse over the task or category name, a pop-up Task Tip displays with a brief description of the task or category. To choose a task, simply click on the task name.

Task Panel

The Task Panel is the main working area for completing a task. Once you select a task, this area displays a help message, information and prompts associated with the task. At the bottom of the task panel are action buttons that allow you to finish the task once you have entered all necessary information. Later sections of this document provide a detailed description of each task's Task Panel.

Status Message Panel

This area displays messages about the application's status, or confirmation messages about a completed task. For example, when you first access a screen you'll see "Ready" displayed, which indicates that the application is ready to process data. When you perform a task such as restarting a problem for a student, you'll see a message confirming that the problem has been restarted.

Understanding the Overall Process

The following scenario describes a typical process for using Teacher's Toolkit to create, maintain and administer a Cognitive Tutor. The scenario is a snapshot of the overallprocess and is not meant to imply a definitive task order. Note that the Task List name associated with each task is shown in parentheses.

Beginning of the Term

  • Add information for any new instructors (Add New Instructor)
  • Create a new class for a particular curriculum and instructor (Create Class)
  • Add or import student names to the new class (Add Student)

Ongoing Student Maintenance

  • Add a new student to a class (Add Student)
  • Transfer a student to a different class (Transfer Student)
  • Change a student's name or password (Edit Student)
  • Delete a student from a class (Delete Student)

Ongoing Instructor Maintenance

  • Add information for a new instructor (Add New Instructor)
  • Change an instructor's name, login or passwords (Edit/Delete Instructor)
  • Delete an instructor from the system (Edit/Delete Instructor)
  • Reassign a different instructor for a class (Change Class Instructor)

Ongoing Curriculum Administratione

  • Restart a problem for a student, or skip to the next problem (Restart/Skip Problem)
  • Start a student at a particular lesson or section within the curriculum (Change Student Placement)
  • View the problem text and its solution (View Problem & Solution)

Ongoing Reporting

  • View the current progress of each student in a class (View Class Roster)
  • Print a detailed progress report for an individual student, or for all students in a particular class (Detailed Student Reports)
  • Print or view a class roster report showing the number of units completed per student, and each student's position within the curriculum. (Class Summary Reports)
  • Print or view a class progress report that graphically displays the number of students per class that are currently working within each unit in the curriculum. (Class Summary Reports)

End of the Term

  • Delete class records and class rosters (Delete Class)
  • Delete data records for instructors who will not be returning next term. (Edit/Delete Instructor)
