lmsHost

The LMS Host is the host name or IP address of the machine that is running the LMS Server process. If LMS is not being hosted remotely, this parameter should not be entered.

Special Case: if the lmsHost value is set to dynamic, then the Cognitive Tutor launcher will prompt the user to enter the name or IP address of the remote LMS server host, as well as the lmsPort.


lmsPort

The LMS Port is the socket port that LMS uses to communicate with the tutor or Teacher's Toolkit. For most installations, this should be set to 60080.

For CITRIX installations, this should be set to dynamic.

schoolID

This is the identifier for your school. For most installations, this value should be Local.

launcherPort

This is the socket port that the Launcher uses to detect if another Cognitive Tutor application is already running. All Cognitive Tutor applications should be using the same Launcher Port value.

For CITRIX installations, this should be set to dynamic.
