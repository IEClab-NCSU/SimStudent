## WEB BASED USER INTERFACE FOR SMART

This is a web user interface for SMART, which stands for Skill Model mining with Automated detection of Resemblance among Texts.
The website has been made with HTML & CSS on the front end and Python on the back end.
Flask framework has been used to run the server. Python has been chosen 
on the back end because it is one of the best languages for Machine Learning applications.
